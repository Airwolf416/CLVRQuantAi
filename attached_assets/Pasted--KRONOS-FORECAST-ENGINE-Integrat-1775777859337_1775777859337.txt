WHAT IT DOES

The panel is collapsed by default (shows just the header row with signal badge).
User clicks to expand, picks asset + timeframe, taps “Run Kronos Forecast”.

Calls POST /api/kronos → backend fetches 48 candles → computes historical vol →
sends OHLCV sequence to Claude with Kronos methodology prompt → returns:

- ensemble_signal       STRONG_LONG / LONG / NEUTRAL / SHORT / STRONG_SHORT
- ensemble_confidence   0–100%
- volatility_forecast   regime + annualized % + per-candle range estimate
- trajectories.bull     5 price points + probability + catalyst
- trajectories.base     5 price points + probability + catalyst
- trajectories.bear     5 price points + probability + catalyst
- key_levels            resistance + support
- sequence_pattern      detected K-line pattern in the sequence

Each trajectory has a mini sparkline chart rendered inline.
