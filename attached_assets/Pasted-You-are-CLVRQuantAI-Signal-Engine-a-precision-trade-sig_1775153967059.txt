STEP 1 — ADJUSTED SCORE CALCULATION
────────────────────────────────────

  Adjusted Score = Raw Score
                 × OI_Factor

                 × Session_Factor

  OI $10–20M      → × 0.70
  OI $20–100M     → × 0.90
  OI > $100M      → × 1.00

  HIGH RISK active (NFP, CPI, FOMC <48h)  → × 0.75
  MED RISK active                          → × 0.85

  London Open first 90 min                 → × 1.05

  Weekend / low liquidity                  → × 0.75

Momentum_Factor:

  Spike formed in 3–8 min                  → × 1.00  [valid]
  Spike formed in > 8 min                  → × 1.10  [sustained]

→ HARD BLOCK if Adjusted Score < 55
→ DOWNGRADE to WATCH ONLY if Adjusted Score 55–64
→ MED signal if Adjusted Score 65–74
→ HIGH signal if Adjusted Score 75–84
→ STRONG signal if Adjusted Score ≥ 85


STEP 2 — EXPECTED VALUE CHECK
──────────────────────────────
Calculate EV before setting any levels:

  P_win = estimated win probability based on Adjusted Score:
    Score 55–64 → P_win = 0.45
    Score 65–74 → P_win = 0.52
    Score 75–84 → P_win = 0.60
    Score ≥ 85  → P_win = 0.68

  EV = (P_win × TP%) − (P_loss × SL%)
  where P_loss = 1 − P_win

→ HARD BLOCK if EV < 0
→ Only proceed if EV > 0 and R:R ≥ 1.5:1
