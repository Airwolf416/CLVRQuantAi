=== BUTTON WIRING ===
The "Upgrade to CLVR Pro" button should call:

const handleUpgrade = (tier) => {
  // tier = 'pro' or 'elite'
  // billingCycle = 'monthly' or 'yearly' (from the toggle state)
  if (!user) {
    // redirect to login with return path
    window.location.href = `/login?returnTo=/checkout?priceId=${PRICE_IDS[tier][billingCycle]}`;
    return;
  }
  if (user.tier === tier) {
    // already on this tier — show toast or redirect to account
    return;
  }
  window.location.href = `/checkout?priceId=${PRICE_IDS[tier][billingCycle]}&plan=${tier}&cycle=${billingCycle}`;
};

Elite button: onClick={() => handleUpgrade('elite')}
