## SECTION 5 — MACRO PRE-FLIGHT (Must Run Before Every AI Call)

Before EVERY AI call (Trade Ideas, Quant Scanner, Basket, Ask AI), fetch macro events and inject them.

### 5A. Create Pre-Flight Endpoint (if not already created)

```typescript
// GET /api/macro/preflight
app.get("/api/macro/preflight", async (req, res) => {
  try {
    // Pull from DataBus (already available)
    const databus = await fetch(`http://localhost:${PORT}/api/databus/status`).then(r => r.json());

    const macroEvents = databus.macroEvents || [];
    const killSwitch = databus.killSwitch || { active: false };
    const now = new Date();

    // Classify events by proximity
    const next2h = macroEvents.filter(e => {
      const eventTime = new Date(e.time || e.datetime);
      const hoursAway = (eventTime - now) / 3600000;
      return hoursAway > 0 && hoursAway <= 2;
    });

    const next4h = macroEvents.filter(e => {
      const eventTime = new Date(e.time || e.datetime);
      const hoursAway = (eventTime - now) / 3600000;
      return hoursAway > 2 && hoursAway <= 4;
    });

    const highImpactKeywords = ['FOMC', 'CPI', 'NFP', 'rate decision', 'nonfarm', 'fed', 'ECB', 'BOJ', 'BOE', 'BOC', 'GDP', 'inflation'];
    const hasHighImpact2h = next2h.some(e =>
      highImpactKeywords.some(k => (e.title || e.event || '').toLowerCase().includes(k.toLowerCase()))
    );

    const status = killSwitch.active ? 'BLOCKED' : hasHighImpact2h ? 'BLOCKED' : next4h.length > 0 ? 'CAUTION' : 'CLEAR';

    res.json({
      timestamp: now.toISOString(),
      status,
      killSwitch: killSwitch.active,
      eventsNext2H: next2h,
      eventsNext4H: next4h,
      summary: status === 'BLOCKED'
        ? `Macro blocked: ${next2h.map(e => e.title || e.event).join(', ')}`
        : status === 'CAUTION'
        ? `Caution: ${next4h.map(e => e.title || e.event).join(', ')} within 4H`
        : 'Macro clear for next 4H',
    });
  } catch (err) {
    res.json({ timestamp: new Date().toISOString(), status: 'CLEAR', summary: 'Preflight check failed — proceeding with caution' });
  }
});
```

```typescript
async function generateBriefWithRetry(system, userMessage, maxTokens, retries = 3) {
  for (let attempt = 1; attempt <= retries; attempt++) {
    try {
      const response = await callClaudeAPI(system, userMessage, maxTokens);
      if (response && response.text) return response;
    } catch (err) {
      console.error(`[Brief] Attempt ${attempt}/${retries} failed:`, err.message);
      if (err.message?.includes('rate_limit') || err.status === 429) {
        // Rate limited — wait longer
        await new Promise(r => setTimeout(r, Math.pow(3, attempt) * 1000));
      } else {
        await new Promise(r => setTimeout(r, Math.pow(2, attempt) * 1000));
      }
    }
  }
  return null; // All retries failed
}
```
